# `src/benchmark.py` — сравнение производительности base vs fast

Модуль запускает замеры времени для:

- `streebog_512(data)` (базовая реализация)
- `streebog_512_fast(data)` (оптимизированная реализация с T-таблицами)

## Что делает `run_benchmark()`

- Для набора размеров входных данных (`BENCHMARK_SIZES`) генерирует `os.urandom(size)`.
- Для каждого размера:
  - проверяет, что `base_hash == fast_hash`
  - подбирает число итераций так, чтобы общее время измерения было не меньше `min_time` (по умолчанию 0.5 сек)
  - измеряет среднее время одного вычисления для base и fast
- Печатает таблицу в консоль и возвращает список словарей `results`.

## График `save_chart()`

`save_chart(results, path=...)` строит 2 подграфика (бар-чарты):

- время (ось Y в логарифмическом масштабе)
- ускорение (отношение fast/base)

Если `matplotlib` не установлен — выводится предупреждение, график не сохраняется.
