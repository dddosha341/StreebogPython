# `src/streebog.py` — базовая (референсная) реализация Стрибог

Модуль реализует **Стрибог (ГОСТ 34.11-2018)** в «понятном/референсном» виде. Это чистая Python-реализация без оптимизаций T-таблицами.

## Ключевые преобразования

Реализованы как отдельные функции:

- `_S(state)` — S-box (подстановка PI) по каждому байту
- `_P(state)` — перестановка TAU (байтовая транспозиция 8x8)
- `_L(state)` — линейное преобразование L (умножение 64-битных групп на матрицу A в GF(2))
- `_LPS(state)` — композиция `L(P(S(state)))`
- `_X(k, a)` — сложение ключа (XOR) по байтам

## Блочный шифр и сжатие

- `_E(K, m)` — 12 раундов конструкции E из стандарта
- `_g_N(N, h, m)` — функция сжатия (Миягучи-Пренеля):
  `K = LPS(N xor h)` и далее `return E(K, m) xor h xor m`

## Потоковый API

Класс:

- `class Streebog(digest_size=512)`
  - `update(data)` — обрабатывает полные блоки по 64 байта, остаток буферизует
  - `digest()` — финализирует: padding, финальные вызовы `_g_N` и, при `digest_size=256`, усечение
  - `hexdigest()` — hex-представление
  - `copy()` — глубокая копия (используется в алгоритмах поиска)

Удобные обёртки:
- `streebog_512(data) -> bytes`
- `streebog_256(data) -> bytes`

## Endianness и усечение 256

Внутренние счётчики/состояния трактуются как **little-endian** массивы байтов. В реализации `streebog_256` возвращаются последние 32 байта (`h[32:]`), что соответствует MSB-части в LE-представлении.
