# `src/constants.py` — константы ГОСТ

Модуль содержит все таблицы и числовые параметры алгоритма **Стрибог (ГОСТ 34.11-2018)**:

- `PI` (π): S-box (таблица нелинейной подстановки), 256 элементов
- `TAU` (τ): перестановка байтов, 64 элемента
- `A`: матрица линейного преобразования L в GF(2), 64 строки по 64-битных значения
- `C`: 12 итерационных констант для функции `E`, каждая — 64 байта
- `IV_512`, `IV_256`: начальные значения (IV) для 512/256-битного варианта
- `BLOCK_SIZE`: размер блока (64 байта)
- `N_BLOCK_BITS`: константа `512` в виде 64-байтового little-endian числа (используется для инкремента счётчика `N`)

## Endianness

Внутреннее состояние — массив из 64 байтов. Для арифметики счётчика `N` и суммы `Σ` используется **little-endian**.

Константы `A` хранятся так, чтобы соответствовать LE-нумерации битов, используемой в реализации:
- бит 0 — это LSB первого байта группы (соответствует `A[0]`)
- бит 7 — MSB первого байта (соответствует `A[7]`)

## Использование

Импортируется остальными модулями (`streebog.py`, `streebog_fast.py`) для вычислений преобразований S/P/L, раундовых ключей и параметров финализации.
