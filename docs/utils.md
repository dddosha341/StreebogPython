# `src/utils.py` — вспомогательные функции

Содержит простые операции, нужные для реализации Стрибог на уровне блоков:

## Основные функции

- `xor_bytes(a, b) -> bytearray`
  - Побайтовый XOR двух блоков одинаковой длины (в контексте проекта это 64 байта).

- `add_mod512(a, b) -> bytearray`
  - Сложение двух 512-битных чисел по модулю `2^512`.
  - Интерпретирует входы как little-endian целые длиной 64 байта.
  - Возвращает 64 байта в little-endian.

## Endianness

И XOR, и сложение используются в реализации с предположением о формате состояния/счётчика: операции над 512-битными значениями трактуются как `little-endian`.
