# Документация по модулям

Проект реализует хэш-функцию **Стрибог (ГОСТ 34.11-2018)** и инструменты для учебной демонстрации коллизий:
- базовая реализация (`streebog.py`);
- оптимизированная версия с T-таблицами (`streebog_fast.py`);
- поиск коллизий для усечённого хэша `MSB48` (`collision_search.py`);
- построение «осмысленной» коллизии для двух BMP-картинок (`meaningful_collision.py`).

Ниже — краткие README по каждому модулю из `src/`.

- `constants.py` — константы ГОСТ
- `utils.py` — вспомогательные операции над 512-битными блоками
- `hasher_interface.py` — общий интерфейс хэшера и фабрика реализаций
- `streebog.py` — базовая (референсная) реализация
- `streebog_fast.py` — оптимизированная реализация с T-таблицами
- `collision_search.py` — birthday attack для `MSB48`
- `meaningful_collision.py` — осмысленная коллизия для двух BMP
- `cli.py` — CLI для вычисления хэша, коллизий и бенчмарков
- `benchmark.py` — сравнение скорости base vs fast
- `__init__.py` — экспорт публичных API
- `__main__.py` — запуск CLI через `python -m src`
