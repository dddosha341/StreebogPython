# `src/streebog_fast.py` — оптимизированная реализация Стрибог (T-таблицы)

Модуль содержит быструю реализацию Стрибог с предвычисленными **T-таблицами**, которые объединяют эффект `S`, `P` и `L` для каждого входного байта.

## Что ускоряется

В базовой реализации каждый раунд вычисляет `L(P(S(x)))`. Здесь это заменяется на:

1) предвычисление `T[k][v]` (k=0..7, v=0..255)  
2) вычисление результата LPS-группы XOR-ом 8 табличных значений

Это сокращает количество дорогих GF(2)/битовых операций до табличных обращений.

## Предвычисление

- `_precompute_tables()` строит 8 таблиц `T[k]`
- `T` вычисляется один раз при импорте модуля

## Потоковый API

- `class StreebogFast(digest_size=512|256)`
  - `update(data)` — потоковая подача
  - `digest()` — финализация с padding и финальными `_g_N`
  - `hexdigest()`
  - `copy()` — глубокая копия состояния

Внутри `update` используется `_g_N_fast` (та же конструкция Миягучи-Пренеля, но с `_LPS_fast`).

## Удобные функции

- `streebog_512_fast(data) -> bytes`
- `streebog_256_fast(data) -> bytes`
