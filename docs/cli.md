# `src/cli.py` — командная строка

Модуль реализует CLI-интерфейс проекта. Используется для запуска:

- вычисления хэша
- поиска коллизии для `MSB48`
- построения осмысленной коллизии для двух BMP
- бенчмарка base vs fast
- запуска контрольных тестов (встроенные selftest)

## Команды CLI

- `hash`
  - `--alg 256|512` — размер дайджеста
  - `--file` — путь к файлу (если не задан — читается `stdin`)
  - `--fast` — использовать `impl="fast"` (T-таблицы)

- `collision`
  - `--out-dir` — куда сохранить результат (`collision.json`)
  - `--seed` — seed PRNG
  - `--max-attempts` — лимит перебора

- `meaningful-collision`
  - `--out-dir` — куда сохранить результат
  - `--max-attempts` — лимит перебора

- `bench`
  - `--chart [path]` — опционально сохранить график (требуется `matplotlib`)

- `selftest`
  - проверяет контрольные векторы для base и fast, плюс сравнение base==fast на случайных данных

## Точка входа

Основная функция модуля — `main()`, создающая `argparse.ArgumentParser` и диспетчеризирующая `func=...` для подкоманд.
